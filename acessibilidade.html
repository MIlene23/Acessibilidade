Acessibilidade

Entendemos que a internet precisa ser acessível para todos. Com isso em mente disponibilizamos ferramentas e recursos para que seja possível criar relacionamentos saudáveis com a web para qualquer pessoa, independentemente de suas dificuldades visuais e, ou, auditivas.



Nossas opções de acessibilidade:



Ao clicar com o cursor do mouse nesses ícones você poderá aumentar ou diminuir o tamanho da fonte dos textos em todo website de acordo com sua preferência.

          

Ao clicar com o cursor do mouse nesse ícone o website irá ativar o modo de contraste





Ao clicar com o cursor do mouse nesse ícone o website irá ativar o modo monocromático, alterando as cores para escala de cinza.





Ao clicar com o cursor do mouse nesse ícone o website irá alterar para suas cores inversas.





Ao clicar com o cursor do mouse nesse ícone o website destacar todos os links da página.





Ao clicar com o cursor do mouse nesse ícone iniciará em uma janela flutuliante onde você poderá interagir com o website através da linguagem de libras utilizando a Suíte VLibras.


* VLibras: A Suíte VLibras consiste em um conjunto de ferramentas, responsável por traduzir conteúdos digitais (texto, áudio e vídeo) para a Língua Brasileira de Sinais - LIBRAS, tornando computadores, dispositivos móveis e plataformas Web acessíveis para pessoas surdas.

Tecnologias Assistivas
  • Read Alound (Extensão para navegadores): Converte o texto da página da Web em áudio. Gratuito.
    • Google Chrome
    • Mozilla Firefox
    • Microsoft Edge
  • NVDA (Windows): Leitor de tela que interage com o sistema operacional e transforma conteúdo textual em fala. Gratuito.
  • Orca (Linux):Leitor de tela e ampliador, oferece recurso de lupa. Gratuito.
  • VoiceOver (iOS): Leitor de tela que vem instalado como recurso de acessibilidade padrão dos dispositivos da Apple. Gratuito.
  • TalkBack (Android): É o leitor de tela do Google incluído em dispositivos Android. Gratuito.
  • DOSVOX (Windows): Software voltado a pessoas com deficiência visual, que se comunica com o usuário através de síntese de voz e o auxilia em diversas tarefas no computador. Gratuito.
